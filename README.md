# README

This is my class scheduler app for UChicago. It was built using HTML5,
CSS3/Bootstrap, and Ruby.

## To use:
 1. Fork to own GitHub Repository
 1. Clone to Desktop
 1. Run: `bundle install`
 1. Run: `rake db:migrate`
 1. Run: `rake db:seed`
 1. Run: `rails server`
 1. Visit localhost:3000
 1. Enjoy!
