# README

This is my class scheduler app for UChicago. It was built using HTML5,
CSS3/Bootstrap, and Ruby.


To use:
1. Fork to own GitHub Repository
2. Clone to Desktop
3. Bundle install
4. run rake:db migrate
5. run 'rake:db seed'
6. run rails servers
7. visit localhost:3000
8. Enjoy
